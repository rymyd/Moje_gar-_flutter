# Moje Garáž (Flutter)

Osobní aplikace pro evidenci a správu vozidel.

## Build instrukce (pro PC)
1. Otevři terminál:2. APK se vytvoří v `build/app/outputs/flutter-apk/app-release.apk`.

## Funkce
- Přidávání, mazání a úprava vozidel
- Uložení dat lokálně pomocí Hive
- Temné pozadí, české popisky
